## 📌 Sobre o projeto

O **Wide Coverage** é uma landing page desenvolvida como desafio prático de Front-end.

O objetivo do projeto foi praticar a criação de uma interface moderna utilizando **HTML5** e **CSS3**, trabalhando estruturação, estilização, posicionamento de elementos e responsividade.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Google Fonts
- Figma

## 💻 Funcionalidades

- Layout moderno e limpo
- Menu de navegação
- Botão de cadastro
- Seção principal com imagem e chamada
- Design responsivo
- Estrutura organizada

## 🎯 Objetivo do desafio
